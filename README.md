# TiO2_Nanotubes_ML
Задача: Прогнозирование морфологии нанотрубок 
 (длина, диаметр, структура) на основе условий синтеза (ток, состав электролита, температура).
Стек и Методология
Инструменты: Python, Scikit-learn, Pandas.
Модели: Сравнение Random Forest и Gaussian Process Regression (GPR).
Выбор: Финальная модель — GPR. Она показала лучшую обоснованность и точность на физико-химических данных.
